## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/SnippetTechie/DevLance.git
cd DevLance
git checkout ui/ux
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```

4. Open browser to `http://localhost:5173`

### Build for Production
```bash
npm run build
```
